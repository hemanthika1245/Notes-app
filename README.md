# 📝 Notes App

A simple and responsive full-stack Notes Application that allows users to create, view, update, and delete notes. This project was built to practice full-stack web development by integrating a frontend with a RESTful backend and a MySQL database.

## 📌 Project Overview

The Notes App helps users organize their notes in a clean and user-friendly interface. It demonstrates the complete CRUD (Create, Read, Update, Delete) workflow using modern web development technologies.

## ✨ Features

* ➕ Create new notes
* 📄 View all saved notes
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 🔄 Real-time updates using REST APIs
* 💾 Store notes permanently in a MySQL database
* 📱 Responsive and clean user interface
